# AI Product Content Generator

AI-powered tool for generating high-converting eCommerce content for Amazon and Shopify sellers.

## Features

- Amazon Listing Generator (Title + Bullet Points + Description)
- AI-powered copywriting
- One-click copy to clipboard

## Tech Stack

- Next.js 14+
- Tailwind CSS
- OpenAI API

## Getting Started

```bash
npm install
npm run dev
```
